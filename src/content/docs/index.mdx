import { Card, CardGrid, LinkCard } from '@astrojs/starlight/components';

Hivemind models services, secrets, configs, networks and volumes through an API
and web console, deploys them with full history and one-click rollback, and
supervises the running fleet live — over a direct connection **or** a
reverse-tunnel agent for clusters with no inbound exposure.

<CardGrid>
  <Card title="Multi-cluster" icon="puzzle">
    Every resource is scoped to a cluster. Switch the active cluster from the UI
    and lists, creates and deploys follow it.
  </Card>
  <Card title="Agent for NAT" icon="rocket">
    A reverse-tunnel agent dials out, so a firewalled cluster needs no open
    ports and never exposes its Docker socket.
  </Card>
  <Card title="Deploy with memory" icon="approve-check">
    Draft, deploy, and roll back. Every deployment is recorded; snapshots make
    point-in-time restore one click.
  </Card>
  <Card title="Single image" icon="seti:docker">
    The Go binary embeds the Angular UI and serves API + UI on one port.
  </Card>
</CardGrid>
